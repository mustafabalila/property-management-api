# Property Management API

## Requirements

- Node.js
- PostgreSQL
- Redis
- `node-gyp` (See: [building argon2](https://github.com/ranisalt/node-argon2#before-installing))
- Optional: `docker` & `docker-compose`

## Development

```bash
$ npm run dev
# run in development

$ npm run start
# run in production

$ npm run lint
# check for errors

$ npm run fix
# fix any errors

$ npm run test
# run tests
```
